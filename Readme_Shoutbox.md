### 🇩🇪 Shoutbox-Mini-Chat-Widget

- **Turbo-basiertes Messaging-Widget**
  - ⭐ Ein ein- und ausblendbares Widget mit Nachrichtenanzeige, Nachrichtenzähler und einem Texteingabefeld
  - 💬 Angemeldete Benutzer können Nachrichten mit bis zu 120 Zeichen in unbegrenzter Anzahl verfassen
  - 💬 Die Anzeige zeigt die letzten 50 Nachrichten in einem scrollbaren Container an (einstellbar, 500 im Speicher)
  - 🔧 Der Administrator kann jede Nachricht löschen, Benutzer können dies nicht
  - 🔧 Der Administrator kann diese Funktion im Reiter „Admin-Einstellungen“ deaktivieren
  - 🌐 8 Sprachen (de/en/es/fr/ja/ko/ru/zh_CN)

- **UX & Design**
  - ⭐ Turbo-basiertes Messaging-Widget
  - ⭐ Stimulus steuert das Öffnen und Schließen
  - ⭐ Shoutbox beim Start ausgeblendet
  - ⭐ Farben für Hell-/Dunkelmodus
  - ⭐ Flüssiges Verhalten beim Öffnen und Schließen
  - ⭐ Verbessertes Verhalten auf Mobilgeräten
  - 🌐 8 Sprachen (de/en/es/fr/ja/ko/ru/zh_CN)
  
### 🇺🇸 Shoutbox-mini-chat widget

- **Turbo-based messaging widget**
  - ⭐ A toggleable widget with a message display, message counter and an text input field
  - 💬 Logged in users can type 120 character messages in unlimited number
  - 💬 Display shows last 50 messages in scrollable container (adjustable, 500 in storage)
  - 🔧 Admin can delete any message, users can't
  - 🔧 Admin can deaktivate this function in Admin-Settings-Tab
  - 🌐 8 languages (de/en/es/fr/ja/ko/ru/zh_CN)

- **UX- & Design**
  - ⭐ Turbo-based messaging widget
  - ⭐ Stimulus controls opening and closing
  - ⭐ Set shoutbox hidden at startup
  - ⭐ Colors for light/dark mode
  - ⭐ Smooth behavior on opening and closing
  - ⭐ Improved behavior on mobile devices
  - 🌐 8 languages (de/en/es/fr/ja/ko/ru/zh_CN)
